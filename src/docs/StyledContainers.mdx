# StyledContainers

## Container
<PropsTable of={Container} />

### Standard Usage
Keep in mind that the media queries are responding to the viewport, not the Docz Playground wrapper (you'll have to resize the browser window to see spacing changes).
<Playground>
  <Container
    spacing={{
      horizontal: {
        sm: '15px',
        md: '30px',
        lg: '50px',
      }
    }}
    mediaQueries={createMediaQueries({
      md: 992,
      lg: 1200,
    })}>
    {_.times(6, () => (
      <p>{faker.lorem.paragraph()}</p>
    ))}
  </Container>
</Playground>
