# FlexibleLayout

<PropsTable of={FlexibleLayout} />


## Notes:
- You can use the FlexibleLayout component and pass it an array of items, or you can use each layout component agnostic of the FlexibleLayout component.
  - When using FlexibleLayout, you can also pass a layout-specific className or style prop via the `itemsProps` prop. It is an array of objects (one for each [layout type](#layout-types)). Example:
  ```javascript
    item: 'type',
    className: '',
    style: {},
  ```
- [Here](#wordpress-acf-json-flexible-content) is the WordPress Advanced Custom Fields (acf-json) data structure that can be used in your own WordPress project along with this FlexibleLayout component.
- LazyLoadImage(s) don't currently work in Docz.
- All layouts do have default props (Docz just doesn't show them b/c they are a shared object).

## Layouts Default Props
```javascript
{
  className: '',
  style: {},
  rowSpace: 60,
  columnSpace: 30,
  breakpoint: 992,
}
```

## Layout Types
```
[
  'one_column',
  'two_columns',
  'two_columns_image_leads',
  'two_columns_stacked',
  'two_columns_three_column_list',
  'two_columns_image_grid',
  'video',
  'content_grid',
  'image_grid',
]
```
